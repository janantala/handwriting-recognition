# Latex seed v0.0.1

Basic structure for latex made documents.


## What is included?

### Tex files
- document.tex - master file
- title.tex - title page
- intro.tex - introduction, table of contents, list of tables, list of figures
- content.tex - main
- appendix.tex - appendices

### Bibliography
- bibliography.bib

### Build scripts
- build-bibliography.sh
- build-text.sh
